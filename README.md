# applets-tools

## 项目安装
```
npm install
```

### 项目开发热重载
```
npm run electron:serve
```

### 项目打包
```
npm run electron:build
```

### 项目功能
```
1、多小程序代码自动上传
2、自动化提交代码审核
3、审核通过代码提交上线
```
